# QA Playwright Framework

Automation testing framework built using Playwright + TypeScript.

## Website Under Test
https://www.saucedemo.com/

----

## Features

- Playwright with TypeScript
- Page Object Model (POM)
- Reusable utilities
- Test data management
- HTML reporting
- Screenshot on failure
- Cross-browser support
- CI/CD ready

----

## Project Structure

qa-playwright-framework/
│
├── tests/
├── pages/
├── test-data/
├── utils/

## Test Scenarios

## Install Dependencies

```bash
npm install
```

## Run Tests

```bash
npx playwright test
```

